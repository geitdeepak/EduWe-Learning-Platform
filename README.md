# 🚀 EduWe Learning Platform

EduWe is a modern **AI-powered learning platform** that helps learners explore technology through curated YouTube tutorials, structured learning paths, and personalized recommendations.

---

## 🌟 Features

### 🎯 Smart Learning Experience
- 📺 Curated YouTube-based learning (no video hosting)
- 🎯 AI-based "Recommended for You"
- ⏯ Continue Watching feature
- 📊 Progress tracking system

### 📚 Learning Management
- 🧭 Learning Paths (Azure, AWS, AI/ML, Python, IoT)
- 🔖 Bookmark (My Learning section)
- 🔍 Search & Category Filtering

### 🎨 UI/UX
- 🌙 Modern dark theme (Gen Z style)
- 📱 Fully responsive design
- ⚡ Smooth and interactive UI

---

## 🧠 AI Features

- Recommendation system based on user activity
- Smart category-based suggestions
- Personalized learning experience
